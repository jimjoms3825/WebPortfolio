Procedural Generation Project

Written in C# in the Unity game engine. Deployed for desktop.

The primary prerogative of this development was to explore techniques for 3D procedural dungeon generation using a custom algorithm which stitches together designer made room objects in unbounded space, producing a highly variable interior environment with unbounded size. The development secondarily consisted of the surrounding game framework to integrate this algorithm into a larger project. This led to the development of a multi-octave perlin noise exterior terrain generator, behavior tree implementation for AI, and much more.
